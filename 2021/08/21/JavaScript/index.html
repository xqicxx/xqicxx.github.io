数据类型

认识六种数据类型

1. Boolean 布尔值

  1. 布尔值有两种：True和false；

2. Null 空值

  1. var a=null;

3. Undefined 未定义

  1. 若不定义变量，则类型值为：undefined;

4. String 字符串

  1. 需要用单引号或者双引号引起来
  2. 在字符串中使用\作为转义字符

5. Number 数值

  1. Number = 整型+浮点数
  2. 最大值为，Number.Max_Value
  3. Infinity为无穷大，+infinity\infinity为正无穷，-infinity为负无穷；
  4. NaN = Not a Number
  5. 二进制数为：0b开头；
    八进制为：0开头；（如果出现020的十进制书，容易误判为2进制，可以强制用 (020，10)强制转为10进制。）
    十六进制为：0x开头

6. Object 对象

前五种为基本数据类型，objct为引用数据类型
可以用typeof来检查数据的类型

4. 逻辑运算符:

  1. ! : 即非运算
    1. 可以给布尔值取反;
    2. 如果是其他类型的则会转化为布尔值;
  2. && : 即与运算
    1. &&两边为true时,返回true;
    2. &&检查到一边为false后,就不再往后运算;
    3. 其它类型的值进行&&运算,会先转换为Boolean值;
    4. 规则:
      1. 第一个为true返回第二个值;
      2. 第一个为false返回第一个值;
  3. || : 即或运算
    1. ||两边一边true时,返回true;
    2. ||检查到一边为true后,就不再往后运算;
    3. 其它类型的值进行||运算,会先转换为Boolean值;
    4. 规则:
      1. 第一个为true返回第一个值;
      2. 第一个为false返回第二个值;

5. 关系运算符

  1. > , >= , < , <= , 用来比较两个值的大小;
  2. 成立则返回true , 否则返回 false ;
  3. 除字符串外,其他的类型的值比较会转换为 Number 值;
  4. 字符串比较的时Unicode编码;

6. 赋值运算符

运算符 功能 例子

*= javascript var a= 1; a *= 3 等于 a = a * 3;
/= javascript var a= 1; a /= 3 等于 a = a / 3;
%= javascript var a= 1; a %= 3 等于 a = a % 3;

7. 相等运算符

  1. == 左右两边的值是否相等,相等则返回true,否则返回false;
    1. 不同类型的比较,会转换为相同类型进行比较;
    2. null == undefined;
  2. != 判断左右两边的值是否不相等,不相等返回true,相等返回false;
    1. 会自动转换类型进行比较;
  3. === 判断两边值是否全等,与 == 类似,但不会进行类型值的转换,类型不同直接返回false;
  4. !== 判断两边值是否全等,与 != 类似,但不会进行类型值的转换,类型不同直接返回true;
  • 注意 :
    • NaN不与任何值相等,包括它本身;
    • NaN可以用isNaN();来判断自身是否为NaN值;

8. 三元运算符

  1. :?
    1. 语发:条件表达式:?语句1:语句2;
    2. 流程:先对条件表达式进行判断,若为true则执行语句1,反之执行语句2;
  • 注意 : 优先级若不确定,则可以运用()来修改他的优先级;

基本数据类型和引用数据类型的区别

  1. 基本数据类型存在于栈内存中，值与值之间独立存在，不影响；

  2. 引用数据类型存在于堆内存中，值会开辟出一个新的空间，而变量在栈内存里保存的是对象地址；

  3. 基本数据类型比较值时，单纯比较值；而引用数据类型比较值时，会比较对象地址，若值一样对象地址不一样，也会返回false；

函数（Function）

认识函数（Function）

  1. 函数一个对象，可以封装一些功能，在需要的时候调用；

函数的创建

  1. 以创建对象的形式；

    1. 可以将要封装的代码以字符串形式传递给构造函数；

    2. 以函数名()来调用；

  2. 使用函数表达式

    1. 先创建一个匿名函数,再赋值;

函数参数

  1. 创建函数时，可以再()内指定一个或多个形参；

  2. 调用函数时，可以再()内传对应的实参；

    1. 解析器不会检查传递的数值，所以要注意是否接收非法的参数，所以要检查；
    2. 解析器不会检查实参的数量，多余的数量不会被赋值；
    3. 实参少于形参，剩下的形参会被赋值为undefined;

  3. 实参可以是对象

    1. 当我们的实参太多时，可以把实参封装到对象里面，再调用；

  4. fun();和fun的区别

    1. fun()调用函数，相当于使用函数返回值；
    2. fun函数对象，相当于直接使用对象；

函数返回值(return)

  1. 设置函数(function)的返回值；

  2. return后的语句都不会执行；

  3. return后不跟值或者不写return就相当于返回undefined;

  4. 语句也可以直接写在return后面，返回他的值；

  5. 语法: return 值;

  6. break,continue,return的区别

    1. break退出循环；
    2. continue跳过当次循环；
    3. return退出函数；

  7. 返回值可以是任意的数据类型，包括对象、函数；

立即执行函数

  1. 写一个匿名函数，在跟它整体加个()再在后面加个();
   (function fun(a,b){
      console.log("a ="+a);
      console.log("b ="+b);
   })(123,456);
   //结果：a = 123;b = 456;
