数据类型

数据类型

基本数据（值）类型

  1. Number
  2. String
  3. Boolean
  4. undefined
  5. null

引用（对象）数据类型

  1. Object对象
  2. Function（一种可以执行的特殊对象）
  3. Array（带有下标并且内部数据有序排列）

null和undefined的区别

  1. undefined为值未定义
  2. null为值为空

null的使用

  1. 在创建对象时先赋一个值null来占位;
  2. 不需要使用此对象时，使用null来将它设置为垃圾对象，让系统回收；

判断类型的方法

typeof

  1. 因为用typeof判断Null、Object、Array都会返回object，所以不能判断这三种;

  2. 能够判断：string、number、boolean、undefined、function

  3. console.log(typeof a);返回的是a数据类型的字符串;

  4. null用===来判断;

===

  1. 与==的区别：不进行数据类型转换的比较;

  2. 可以直接判断null和undefined;—>他们只有一个值，其他有多个值。(如：布尔值有两个：true和false);

instanceof

  1. 判断某个构造函数的prototype属性是否出现在某个实例对象的原型链上。
  2. 语法：A instanceof B;
