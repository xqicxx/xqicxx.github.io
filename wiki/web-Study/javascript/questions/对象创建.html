对象的创建

Object构造函数

  1. 用法：对象内部数据不确定;
  2. 缺点：代码太多;
  3. 用工厂方法;

对象字面量

  1. 用法:对象内部数据应景确定;
  2. 缺点:多个创建代码重复、多;
  3. 用工厂方法;

工厂方法(返回对象字面量)

  1. 用法:需要创建多个对象;
  2. 缺点:都是Object类型的; console.log(p1 instanceof creatPerson); // false
  3. 解决:用构造函数;

构造函数

  1. 用法:需要创建多个对象、且有自己的类型;
  2. 缺点:创建对象后会又多个重复代码;
  3. 解决:与原型一起使用;
